Miscellany

Various works in progress.

FPBox is a beagleboard listening to and fingerprinting live audio, then scrobbling it. After identifying a track it will download analysis and metadata for that track in order to modulate its own VU meter and display the track name on the LCD display.

Remix is a dynamic instrument that synthesizes new sounds from ambient noises and then allows a user to play them. It's built on a mini ITX computer inside a laser cut enclosure. The only interface is the 4x4 RGB button pad.

A usb controller for an LED Matrix. This circuit was printed in an epson injet printer that I modified to print PCBs. I then baked and etched it.
