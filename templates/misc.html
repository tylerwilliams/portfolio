Miscellany

Various works in progress.

FPBox is a beagleboard listening to and fingerprinting live audio, then scrobbling it to last.fm. After identifying a track it will download analysis and metadata for that track in order to modulate its own VU meter and display the track name on the LCD display. I designed the controller and software for the meter and display, which is available here.

Remix is a dynamic instrument that synthesizes new sounds from ambient noises and then allows a user to play them. It's built on a mini ITX computer inside a laser cut enclosure. The only interface is the 4x4 RGB button pad and a microphone.

A usb controller for an LED Matrix. This circuit was printed in an epson inkjet printer that I modified to print PCBs. I then baked and etched it.
